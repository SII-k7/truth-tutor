# v0.1.1 Focus

## Product shift

From:
- a general diagnosis-first learning coach

To:
- a general diagnosis-first learning coach
- plus a dedicated paper-reading module
- plus an alphaXiv recovery intake mode

## Why this matters

Paper reading is not just another study task.
It has its own failure modes:
- reading above footing
- notation overload
- derivation avoidance
- confusing familiarity with understanding
- not knowing what order to reread in

That is why it deserves a separate module.

## v0.1.1 deliverables

- dedicated paper-reading prompt profile
- alphaXiv recovery prompt profile
- CLI aliases for both
- better examples
- better tests
- updated launch copy
