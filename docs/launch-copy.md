# Launch Copy

## GitHub README hero

**Truth Tutor is a diagnosis-first learning coach for people who do not need comfort — especially graduate students and researchers stuck on papers.**

## ClawHub description

Diagnose why a learner does not understand a topic, concept, or paper, then deliver blunt feedback plus an actionable improvement plan. Includes a dedicated paper-reading mode and an alphaXiv recovery mode for users who already asked for an explanation and still do not get it.

## Social post (Chinese)

我把一个一直想要的工具做出来了：**Truth Tutor**。

它不是那种“我来给你通俗解释一下”的温柔型 AI，核心目标只有一个：
**直接指出你为什么没看懂、你缺什么基础、下一步该补什么。**

这次还专门把**论文阅读**单独拎成了模块，适合研究生/博士生读 paper 时卡在方法、公式、实验、前置知识的场景。

另外还做了一个 **alphaXiv recovery mode**：
如果你已经问过 alphaXiv，但还是“看起来懂了、其实没懂”，Truth Tutor 会直接诊断：
- 是你基础不够
- 还是问题问歪了
- 还是 explanation 的抽象层级不对
