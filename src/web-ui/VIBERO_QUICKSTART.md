# 🚀 Vibero 功能快速启动指南

## 立即开始

### 1️⃣ 启动应用
```bash
cd /Users/zhukeqi/.openclaw/workspace/truth-tutor-repo
npm start
```

### 2️⃣ 打开浏览器
访问应用地址（通常是 `http://localhost:3000`）

### 3️⃣ 切换到 Vibero 模式
**方法 A**: 点击右上角的 **"✨ Vibero 沉浸模式"** 按钮  
**方法 B**: 按键盘快捷键 `Ctrl+V` (Mac 用户按 `Cmd+V`)

### 4️⃣ 加载 PDF 文档
- 在搜索框中输入论文 ID 或标题
- 或者直接粘贴 arXiv 链接

### 5️⃣ 体验 Vibero 功能

#### 📚 双侧导航面板
- 左右两侧显示文章脉络
- 点击章节标题快速跳转
- 当前章节自动高亮

#### 📌 便签式标注
- 彩色便签标记重要内容
- 悬停查看翻译提示
- 点击打开 AI 解答侧边栏

#### 🎮 浮动控制
- 右下角显示上一章节/下一章节按钮
- 快捷键: `Ctrl+P` (上一章节), `Ctrl+N` (下一章节)

#### 📊 阅读进度
- 导航面板底部显示进度条
- 实时更新阅读百分比

## 🎯 核心功能演示

### 标注交互流程
1. 鼠标悬停在便签标注上 → 显示翻译提示
2. 点击便签标注 → 打开 AI 解答侧边栏
3. 查看详细解释、参考资料
4. 点击侧边栏外部或 × 按钮关闭

### 导航交互流程
1. 查看左右导航面板的文章脉络
2. 点击章节标题跳转到对应页面
3. 使用浮动按钮或快捷键在章节间快速切换
4. 观察阅读进度实时更新

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+V` / `Cmd+V` | 切换 Vibero 模式 |
| `Ctrl+P` / `Cmd+P` | 上一章节 |
| `Ctrl+N` / `Cmd+N` | 下一章节 |
| `Ctrl+D` / `Cmd+D` | 切换深色模式 |

## 🔧 故障排除

### 问题: 切换按钮不显示
**解决**: 检查浏览器控制台是否有错误，确保所有组件文件都已加载

### 问题: 导航面板不显示
**解决**: 
1. 确保已切换到 Vibero 模式
2. 检查是否已加载 PDF 文档
3. 刷新页面重试

### 问题: 标注不显示
**解决**:
1. 确保 `test-annotations.json` 文件存在
2. 检查标注数据格式是否正确
3. 打开浏览器控制台查看错误信息

### 问题: AI 解答显示"加载失败"
**说明**: 当前使用模拟数据，需要连接真实 API
